# Reference Comparison Data

This directory stores the canonical external comparison files used by the
protected paper-reproduction examples. These files were copied into the
repository so the comparison data forms part of the permanent scientific
record for `ex1` through `ex7`.

These files are not part of the small modular regression test suite. They are
used only by the heavier paper-reproduction workflows.

## Layout

- `yspec/`
  YSpec comparison traces used by `ex1`, `ex2`, `ex3`, and `ex7`
- `specnm/`
  SpecNM comparison traces used by `ex1`, `ex2`, `ex3`, and `ex7`
- `mineos/bolivia/`
  MinEOS comparison traces used by `ex2`
- `mineos/noheader/`
  MinEOS comparison traces used by `ex3` and `ex7`

## Provenance

The files below were copied from the previously external sibling-directory
workflow locations. The examples were updated to read these in-repo copies via
paths relative to the build directory, preserving the same runtime lookup model
while removing the dependency on sibling repositories.
