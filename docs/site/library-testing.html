Library Testing

The release test suite focuses on small, synthetic, local checks that narrow failures quickly. It is designed to catch regressions in parsing, helper logic, normalization, filtering, writers, and small SEM components without relying on full waveform comparisons.

Test Categories

  • Unit: parser rules, value validation, helper functions, normalization, and file-format readers.
  • Component: output writers, filter option validation, and minimal SEM/local-to-global checks.
  • Smoke: t1 runtime execution.
  • Paper validation: environment-specific reproduction workflows for ex1 through ex7.
