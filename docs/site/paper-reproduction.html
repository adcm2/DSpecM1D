Paper Reproduction Examples

Important: ex1 through ex7 are preserved as paper-reproduction workflows. Their external comparisons are intentional and must remain intact for scientific reproducibility.

These examples are not the main regression test suite. They are validation workflows for the supported reproduction environment used to compare against YSpec, MinEOS, and SpecNM outputs.

Current External Inputs

  • ex1, ex2, ex3, and related plotting flows read comparison outputs from sibling YSpec and SpecNM directories.
  • ex7 additionally depends on MinEOS comparison traces.
  • These dependencies are part of the paper workflow and should be treated as deliberate release requirements for reproduction, not accidental coupling.

Verification Model

The protected examples should continue to build and run in the supported environment with their current output flow unchanged. Day-to-day regression catching should instead happen in the modular library tests.
