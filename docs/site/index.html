Release Documentation

DSpecM1D

A C++23 spectral element library for synthetic seismograms in a spherically symmetric Earth model, with protected paper-reproduction examples and a separate self-contained regression test strategy.

C++23 Sparse SEM Frequency Domain Pinned CMake Dependencies

Recommended Entry Points

Preferred Release API

InputParametersNew, SPARSESPEC::SparseFSpec, DSpecM::FilterOptions, DSpecM::applyFilter, and the output-writer helpers are the intended public path for new users.
