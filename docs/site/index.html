Release Documentation

DSpecM1D

A C++23 spectral element library for synthetic seismograms in a spherically symmetric Earth model, with a release workflow that distinguishes paper-reproduction examples from modular library tests.

Recommended Entry Points

InputParametersNew, SPARSESPEC::SparseFSpec, DSpecM::FilterOptions, DSpecM::applyFilter, and the output-writer helpers are the preferred release-facing API.
