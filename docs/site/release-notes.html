Release Notes

This release-readiness pass introduces a modular test suite, a self-contained documentation website, Doxygen support, exported CMake package metadata, and a clearer distinction between tutorial smoke tests and paper-reproduction validation examples.

Release Gates

  • Build the library and examples.
  • Run modular unit/component tests.
  • Run the self-contained t1 smoke test.
  • Run protected paper-reproduction workflows in the supported environment.
  • Build the website and API docs.
