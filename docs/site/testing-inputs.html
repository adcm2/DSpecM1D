Parser And Workflow-Context Tests

These tests protect the small but high-leverage entry points that turn parameter files into usable solver context.

What Is Covered

  • InputParser helpers for skipping comments, parsing scalars, and reading latitude/longitude pairs.
  • The legacy InputParameters parser on valid input and on invalid geographic input.
  • InputParametersNew path resolution, normalization-factor selection, and setter clamping.
  • SpectraRunContext construction and accessor behavior.

Why These Tests Matter

Small parser or context regressions can make every example fail in confusing ways, so these tests are designed to catch malformed input handling and context-assembly mistakes as early as possible.
