Installation

Requirements

DSpecM1D requires a C++23 compiler, CMake 3.24+, FFTW3, and OpenMP. The remaining dependencies are fetched automatically from pinned revisions.

cmake -S . -B build/release -G Ninja
cmake --build build/release --parallel

Quick Start

The self-contained tutorial executable is t1. It is the default runtime smoke test because it does not require the external comparison datasets used by the paper examples.

cmake -S . -B build/dev -G Ninja -DDSPECM1D_BUILD_TUTORIALS=ON
cmake --build build/dev --target t1
./build/dev/bin/t1

Installed Package

The release includes install() and exported CMake targets. Downstream projects can consume the installed package with find_package(DSpecM1D REQUIRED) and link against DSpecM1D::DSpecM1D.
