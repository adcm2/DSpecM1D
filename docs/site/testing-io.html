Reference I/O And Output-Writer Tests

These tests keep the comparison-data plumbing small and local so format or truncation mistakes are caught close to their source.

Reference Readers

  • YSpec loading is checked for truncation and zero padding.
  • SpecNM loading is checked for semicolon parsing and short-input padding.
  • MinEOS loading is checked for amplitude scaling and truncation to the shortest component.
  • The bundled YSpec + MinEOS helper is checked directly.

Output Writers

  • Two-way and three-way frequency writers are checked for layout, truncation, and failure on bad output paths.
  • Time writers are checked for output-window truncation, shortest-input truncation, and bad-path handling.
  • InputParametersNew convenience writer overloads are checked directly.
